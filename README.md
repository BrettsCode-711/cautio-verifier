# Cautio Verifier (Public Trust Layer)

**Trust, but verify.**

This is the client-side tool for verifying Cautio proofs. It runs entirely in your browser.

### How to Use
1. Download `index.html`.
2. Open it in any browser (Chrome, Safari, Edge).
3. Drag and drop your `.proof.json` file.
4. View the result immediately.
